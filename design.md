# KNZ Academy - Visual Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a365d) - Conveys trust, professionalism, and academic excellence
- **Secondary**: Warm Coral (#ff6b6b) - Adds energy, enthusiasm, and approachability
- **Accent**: Soft Sage (#68d391) - Represents growth, learning, and success
- **Neutral**: Cool Gray (#f7fafc) - Clean, modern background that doesn't compete with content
- **Text**: Charcoal (#2d3748) for high contrast readability

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings that conveys academic sophistication
- **Body Font**: "Inter" - Clean, highly readable sans-serif optimized for digital screens
- **Accent Font**: "Poppins" - Friendly, rounded sans-serif for buttons and interactive elements

### Visual Language
- **Academic Excellence**: Clean, structured layouts that reflect organized learning
- **Modern Innovation**: Contemporary design elements showing we're forward-thinking
- **Approachable Expertise**: Warm colors and friendly typography that make learning accessible
- **Trust & Reliability**: Consistent design patterns that build confidence in our services

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and element animations
- **ECharts.js**: Interactive progress charts and academic performance visualizations
- **Typed.js**: Dynamic text effects for hero headings
- **Splide.js**: Smooth carousels for testimonials and course showcases
- **p5.js**: Creative background effects and interactive learning elements
- **Pixi.js**: Advanced visual effects for hero sections

### Animation & Effects
- **Hero Text Animation**: Typewriter effect with gradient color cycling
- **Progress Indicators**: Animated circular progress bars for student achievements
- **Card Interactions**: Subtle 3D tilt effects on hover with shadow depth
- **Background Motion**: Gentle floating particles representing knowledge and growth
- **Scroll Animations**: Smooth fade-in reveals as content enters viewport
- **Button Micro-interactions**: Color morphing and gentle scaling on hover

### Header & Navigation Effect
- **Sticky Navigation**: Smooth background blur effect when scrolling
- **Logo Animation**: Subtle rotation and scale on page load
- **Menu Transitions**: Slide-in animations with staggered item reveals
- **Active States**: Animated underlines that follow navigation selection

### Interactive Components Styling
- **Calendar Widget**: Clean grid with soft shadows and hover highlights
- **Progress Trackers**: Animated bars with gradient fills and milestone markers
- **Resource Cards**: Elevated design with preview overlays and bookmark animations
- **Message Interface**: Chat-like bubbles with smooth typing indicators

### Background & Layout
- **Consistent Background**: Soft gradient from light gray to white throughout
- **Section Differentiation**: Subtle geometric patterns and side decorative elements
- **Grid System**: 12-column responsive grid with proper spacing
- **Content Hierarchy**: Clear visual hierarchy using typography scale and spacing

### Mobile Responsiveness
- **Touch-Friendly**: All interactive elements sized for mobile interaction
- **Readable Typography**: Optimized font sizes for various screen sizes
- **Simplified Navigation**: Collapsible menu with smooth transitions
- **Performance Optimized**: Reduced animations on mobile for better performance

### Accessibility Features
- **High Contrast**: All text meets WCAG AA standards (4.5:1 ratio)
- **Focus Indicators**: Clear visual focus states for keyboard navigation
- **Alt Text**: Descriptive alternative text for all images and icons
- **Screen Reader Support**: Proper semantic HTML and ARIA labels

This design system creates a professional yet approachable educational platform that builds trust with parents while engaging students with modern, interactive elements.
